# Spinning Diamond in Unity
This project allows a diamond to spin around its own axis using the Unity game engine.

## Requirements
* Unity 2020.3.0f1 or newer
* Adequate hardware to run Unity on your computer
## Installation
1. Clone or download this repository.
2. Open Unity.
3. Import the project files into Unity.
4. Click on the "Diamond" game object and press the "Play" button to run the game.
